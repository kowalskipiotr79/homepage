Specyficznosc stylow - style mogą być zdefiniowane na wiele sposobów (w wielu miejscach):

Nazywa się to Specyficznością stylów i w oparciu o to przeglądarka ustala jaki styl będzie de facto zastosowany jeżeli dany element strony podlega pod wiele różnych stylów.
Poniżej sposoby na stylowanie wg priorytetów począwszy od najwyższego:
- !important - dopisany w ponizszy sposób nadpisuje wszystkie inne style: a { color: darkorange !important; }
- style - jest to atrybut stosowany do stylowania inline, czyli na poziomie elementu. Przykład zastosowania: <h1 style="color:tomato;">
- #name - stylowanie po id obiektu. Przypisujemy do obiektu na stronie jakies ID i potem w stylach możemy się do niego odnieść za pomocą tego ID. Jest tak, że wiele obiektów może mieć takie samo ID, ale nie jest to wskazane i jest to zła praktyka. Przykład zastosowania:
Ustawiamy id dla jakiegoś elementu: < h1 id="naglowek1" >
W stylach odowłujemy się do niego za pomocą tego ID: #naglowek1 { color: black; }
- .name - stylowanie po klasie obiektu. Przypisujemy obiektudo jakiejś klasy obiektów i potem w stylach możemy się do niego odnieść za pomocą tej klasy. Jest tak, że wiele obiektów może być przypisanych do danej klasy i właściwei to jest cel tego podejścia aby grupować obiekty w klasy i przypisywać im grupowo jakieś style. Przykład zastosowania:
Ustawiamy klasę dla jakiegoś elementu lub elementów: < h1 class="naglowki" > < h2 class="naglowki" >
W stylach odowłujemy się do nich za pomocą tej klasy: .naglowki { color: lemonchiffon; }
- element - tutaj definiujemy style na poziomie typu elementu. Np:
a { color: yellow; }
Można też zagnieżdżać:
nav a { color: khaki; }
To zagnieżdżenie oznacza, że wszystkie znaczniki typu <a> wewnątrz znaczników typu <nav> będą ostylowane w dany sposób.
Przy czym nav a { color: khaki; } jest bardziej specyficzne niż a { color: yellow; } więc ma większy priorytet i to ono determinuje styl tagu <a> wewnątrz <nav>
Natomiast z kolei takie coś:
html body h1#main-header { color: yellow; }
ma przewagę nad:
body .section #main-header { color: red; }
